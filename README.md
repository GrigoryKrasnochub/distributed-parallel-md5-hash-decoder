# distributed-parallel-md5-hash-decoder
client-server application that decode md5 hash

It's study project so i haven't motivation to write big description. 

So it have multiple settings, to make your decode md5 hash expirience unforgettable and uniq. (it's work, it's crush, have fun!)

Start server, start client, set word and wait... Please, stop...

## Comands list

**SHARED**  
i - info  
ss - show/set settings; full format {ss -ip hostIp -p hostPort -l letters -c maxCalcClients}  
**CLIENT**  
c - connect as client; full format {c -ip hostIp -p hostPort}
d - disconect as client  
**SERVER**  
s - start server; full format  {s -p port}  
sd - start decoding; full format {sd yourWordToHashAndDecode}
